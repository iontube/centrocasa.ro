# Lista de colectare eMAG — centrocasa

Cum: intri pe www.emag.ro, cauti termenul din coloana "Cauta pe eMAG",
in panoul verde jos-dreapta pui ETICHETA + MIN RECENZII, apoi Start.
Pagini: lasa 15 (default). Junk-ul din carusele il curat eu automat dupa scanare.

REGULA min recenzii: produse de nisa/scumpe -> 0 (altfel nu trimite nimic).
Produse populare -> 5-15 (ca sa taie junk).

## GATA
- [x] panouri-solare
- [x] saune
- [x] uscatoare-rufe
- [x] mobila  (dormitor + living + bucatarie, toate cu eticheta "mobila")
